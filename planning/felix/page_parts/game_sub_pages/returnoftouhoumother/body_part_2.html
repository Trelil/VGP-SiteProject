Why You Need Japanese Mode

Return of Touhou Mother, being a Japanese game, uses Japanese file names for many of its resources. If not run in Japanese mode, it won't be able to properly find these files and you'll get errors. While I do have a much more streamlined process for getting RPG Maker 2000 games to work with no hassle now, I can't be bothered to do it on the demo. Sorry. But I will for the final version!

Running in Japanese Mode

If you don't have them already, you'll need the East Asian Language Pack files. If you have a Windows install CD, I believe they're on there. If not, well, the internet is pretty big. To install them, go to Control Panel -> Regional and Language Options -> "Languages" Tab. Check the "Install files for East Asian languages" box. I can't check exactly what happens next, since I already have them installed, but I think it will let you install them either from a disk or from a folder.

Once you have Japanese support, there are two options for running a program in Japanese mode.

The first option is more "temporary," you could say. It's using AppLocale, a program from Microsoft that runs programs in a specific language. Run it and choose to "Launch an application." Browse for the .exe. Use whatever arguments you want, I guess - just leave it blank otherwise. If you just installed the East Asian Language Packs, Japanese should be the very last option in the drop-down box. And since they shouldn't be squares anymore, you should also be able to check for this: 日本語 I suggest creating a shortcut. It'll appear in the AppLocale folder, and you can move it wherever you want it. I'd have gone crazy if I had to do the above every time I wanted to test. After clicking Finish, and anytime you click the shortcut thereafter, the game will run in Japanese mode.

The second option is more permanent, but might cause changes in other programs. Go to the "Advanced" tab in Regional and Language Options. The language in the drop down box at the top will automatically be used for non-Unicode programs. Changing it shouldn't be a huge deal, and is nice if you have a lot of programs you want run in Japanese, but like I said, it might mess with a few things. Once it's been changed, you can just run the .exe normally and it should run correctly.
